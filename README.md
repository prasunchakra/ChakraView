# ChakraView

**Vibe Audit Project** — Detect vulnerabilities in web applications.

ChakraView scans web applications for security vulnerabilities based on the OWASP Top 10 and beyond. It provides actionable insights to help developers and security teams harden their applications.

## Features (Planned)

- OWASP Top 10 vulnerability detection
- Custom vulnerability rule engine
- Detailed scan reports with remediation guidance
- CLI-first interface with rich output
- Extensible plugin architecture

## Getting Started

### Prerequisites

- Python 3.10+

### Run

```bash
python main.py
```
